# Portal Web - 数据中台前端

数据中台管理系统前端，基于 Vue 3 + TypeScript + Pinia + Element Plus 构建。

## 功能模块

- 用户管理
- 角色管理
- 菜单管理
- 部门管理
- 字典管理
- 操作日志

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器 (http://localhost:9000)
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- Vue 3.5
- TypeScript 5.7
- Pinia 2.1 (状态管理)
- Vue Router 4.3
- Element Plus 2.9
- Axios 1.7
- Vite 5.2
