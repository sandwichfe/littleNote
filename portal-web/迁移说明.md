## 启动项目

```bash
cd portal-web
npm run dev
```

访问地址：http://localhost:9000

## 环境配置

- 开发环境：`.env.development`
- 生产环境：`.env.production`

后端 API 地址在环境变量 `VITE_API_BASE_URL` 中配置。
